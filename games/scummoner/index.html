Scummoner

Scummoner is a sandbox game where you summon things to ruin lives.

Made in 72 hours for Ludum Dare 55!

CONTROLS

  • WASD / Arrow Keys - Move / Navigate menus
  • Shift - Run
  • Left Click - Summon
  • Number Keys / Scroll Wheel - Select summon
  • Z / Enter - Confirm
  • P - Pause
  • F / Escape - Enter / Exit fullscreen
